Ninja Platformer

by: Lucas Han (bloobglob)

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decided to make this game because I have made other platformers on a different program called scratch, so I thought it would be fun to make one with Java instead of block coding. I also thought that it might be easier to make than other games because I've done it before and can use the same coding concepts.

2. What I Learned

The feedback that I got was that the character looked to plain, so I should change it to something else. I changed it into a ninja. Another piece of feedback that I got was that the game moved to fast, so I made the player move slower.

3. Architecture of My Game

I have 7 different classes. One program to run the whole thing, one program to set up the thing, one program to manage the thing, and the others were the player, lava, walls, and the ground. They interact with each other by having array lists of other objects such as lava, walls, and the ground because there might be more than one wall, or ground.

4. Technical Implementation

The aspect of my game that I am most proud of is the physics. I feel like it runs really smoothly and doesn't glitch into walls or the ground or anything like it. The gravity is more realistic because it pulls down harder and harder, not just at a constant rate.

5. Most Challenging Part

The most challenging part of this project was the object detection. At first, I thought that I could only use one ground object, but I realized that I needed to have 2, the ground and the wall. Another part that was hard was that I thought that I could make multiple rectangles in the different classes, but it didn't work, so I had to make multiple classes (using array lists) instead.

6. Future Fixes and Updates

If I were to continue working on this game, I would definitely add more levels and maybe add trampolines or enemies. I would also probably tweak the physics a little bit so that it could fit better to the game.
